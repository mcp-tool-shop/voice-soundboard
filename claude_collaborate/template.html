How to Create a New Environment

  1. Copy template.html to a new file (e.g., chess.html)
  2. Customize the header, content, and styling
  3. Add your interactive functionality with JavaScript
  4. Register it in index.html sidebar
  5. Test it in Claude Collaborate!
